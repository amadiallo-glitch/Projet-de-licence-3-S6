🚀 DÉMARRAGE DU PROJET
═════════════════════════════════════════════════════════════════════

OPTION 1 - Installation Automatique (Windows):
  1. Double-cliquez sur: setup.bat
  2. Attendez la fin
  3. Lancez: npm run dev

OPTION 2 - Installation Manuelle:
  1. npm install
  2. npm run dev

Le site s'ouvrira sur: http://localhost:5173

═════════════════════════════════════════════════════════════════════

  npm run preview         → Prévisualiser le build
